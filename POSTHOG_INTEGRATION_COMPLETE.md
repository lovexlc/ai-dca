## 🎯 启用 PostHog 仅需 3 步

### Step 1: 注册账号
访问 https://posthog.com/ 免费注册

### Step 2: 获取 API Key
创建项目后复制 API Key（格式：`phc_xxxxx`）

### Step 3: 配置环境变量
```bash
# 编辑 .env.local
VITE_POSTHOG_API_KEY=phc_你的真实KEY
```

重启应用，完成！
