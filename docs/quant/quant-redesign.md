# 量化研究页面重新设计方案

## 设计原则

基于用户需求：
- ✅ **现代简约**：保持浅色系，优化视觉层次
- ✅ **交互式图表**：可缩放、过滤、切换视图
- ✅ **宽松布局**：强调核心指标，增加留白
- ✅ **全面改进**：配置区、回测展示、实盘监控、导航结构

## Tab 1: 策略配置（改进版）

### 当前问题
- H/L ETF 输入是 textarea，不够直观
- 阈值配置缺少说明
- 没有实时预览

### 新设计：标签式输入
```
┌─────────────────────────────────────────────────┐
│ H 高溢价 ETF（卖出方）                          │
│ ┌──────┐ ┌──────┐                              │
│ │159513│ │513100│ [+ 添加]                     │
│ │  ×   │ │  ×   │                              │
│ └──────┘ └──────┘                              │
│                                                 │
│ L 低溢价 ETF（买入方）                          │
│ ┌──────┐                                        │
│ │159501│ [+ 添加]                               │
│ │  ×   │                                        │
│ └──────┘                                        │
└─────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────┐
│ 规则 A：卖 H 买 L                               │
│ 溢价差 ≤ [3.0] % 时触发                        │
│ 说明：高溢价ETF相对低溢价ETF的差价缩小到3%以内   │
│                                                 │
│ 规则 B：卖 L 买 H                               │
│ 溢价差 ≥ [1.0] % 时触发                        │
│ 说明：低溢价ETF相对高溢价ETF的差价扩大到1%以上   │
└─────────────────────────────────────────────────┘

[保存配置]  [运行回测]
```

## 技术实现

### 使用的技术栈
- React + Hooks
- Recharts（交互式图表库）
- Tailwind CSS（保持现有样式系统）
- Lucide Icons（保持现有图标）

### 新增组件
1. `TagInput` - 标签式输入组件
2. `InteractiveChart` - 交互式图表容器
3. `MetricCard` - 大号指标卡片
4. `TabNavigation` - Tab 导航组件
5. `RealTimeSignal` - 实时信号卡片
