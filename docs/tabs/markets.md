# 行情中心

行情中心用于查看 A 股基金和美股自选行情、保存筛选视图，并在选中标的后查看图表、对比、预警和回测。

## 页面结构

### PC 端

- 顶部保留 **A 股行情**入口；美股标的仍可通过搜索或自选列表进入。
- 行情分组：切换场内、场外和自定义分组；自定义分组可重命名或删除。
- 工具栏：搜索、刷新、列设置、筛选、排序、更多功能、卡片/表格视图。
- 表格工具：表头排序和分页；列显示通过顶部“列设置”配置。
- 点击行情行后进入详情；桌面详情保留左侧监控列表，可继续切换标的。

### 移动 Web / App

- 默认显示卡片列表，可切到横向表格。
- 工具栏提供筛选、排序和自定义卡片内容。
- 移动 Web 顶部支持刷新、搜索和保存当前视图；原生 App 只展示当前平台支持的操作。
- 进入详情后使用“行情列表”返回，不需要依赖浏览器返回键。

## 常见操作

### 搜索并加入自选

1. 点搜索按钮，输入代码或名称。
2. 点结果右侧“加入自选”。
3. 已加入的结果会立即变为禁用状态，列表同步更新。
4. PC 端使用搜索框右侧关闭按钮退出搜索。

### 配置列和卡片

1. 点“列设置”或“自定义”。
2. 勾选字段、调整分析指标顺序或列宽。
3. 点“完成”后一次性保存；点“取消”或遮罩不会保存草稿。
4. “重置”只重置当前草稿，仍需点“完成”确认。

隐藏的增强列不会触发对应数据请求。缺少轻量缓存时显示 `—`，不会从列表页同步请求 K 线或净值历史兜底。

### 筛选、排序和视图保存

- 筛选弹层中的“查看结果（N）”会随当前草稿实时更新。
- 同一筛选类别的多个选项按 OR 处理，不同类别按 AND 处理。
- PC 列显示通过顶部“列设置”配置；移动端顶部支持保存当前视图。
- 移动端的筛选、排序和列设置会自动保存到当前行情分组；顶部保存按钮会确认本机已保存，并提示账号同步能力。

### 更多功能

- **导出数据**：导出当前筛选结果为 UTF-8 CSV，逗号和引号会正确转义。
- **只看自选**：筛选当前自选列表中的标的。
- **只看提醒**：只显示已配置行情提醒的标的。
- **指标说明**：显示颜色和缺失值口径。
- **分享页面**：复制当前页面 URL，并显示成功或失败反馈。
- **切换视图**：在卡片和表格之间切换。

## 标的详情

详情页支持：

- 添加/移除自选、设置市场预警。
- 加入持仓、设置买入计划、定投；没有持仓时卖出按钮保持禁用。
- 面积图、点线图、柱形图。
- A 股基金价格、净值和溢价参数。
- MA5/10/20/60、BOLL 指标。
- 最多 3 个对比标的。
- 1 天至最大区间，以及自定义日期。
- 策略回测侧栏。
- 场内基金资金/年报，美股财报/财务等按标的类型展示的详情标签。

场外基金对比在没有 K 线时使用 NAV 历史；只有 K 线和 NAV 都不可用时才显示“无数据”。

## 数据与缓存边界

- 列表只请求 `/quotes`、必要的列表增强接口和已缓存小元数据。
- 列表不会批量请求 K 线、净值历史、财务报表、雪球详情或 AI 摘要。
- 选中标的后才请求 K 线、净值历史、财务和对比序列。
- 场内基金高点下跌只使用缓存的日 K 派生高点；缺失时显示 `—`。
- 场外基金已有 quote 价格/净值时，不额外请求净值快照。
- 申购限额只在场外列表需要展示时请求。

## 常见问题

**Q：为什么某些分析字段显示 `—`？**
A：对应轻量缓存当前不可用。列表不会为了补一个非核心字段同步拉取重数据。

**Q：点“取消”后列设置为什么没有变化？**
A：这是预期行为；列设置使用草稿，只有“完成”才提交。

**Q：美股行情为空？**
A：当前自选列表没有美股代码。搜索并加入美股标的后即可显示。

**Q：分享按钮没有写入剪贴板？**
A：浏览器可能拒绝剪贴板权限，页面会显示失败提示。允许站点剪贴板权限后重试。
