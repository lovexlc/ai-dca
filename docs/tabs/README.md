# 各 Tab 使用手册

这里是 ai-dca 每个标签页的**用户使用说明**：怎么操作、常见问题怎么解决。
实现细节请看 [`docs/architecture/`](../architecture/)。

| Tab | 它能干什么 | 常用操作 |
| --- | --- | --- |
| [持仓总览](./holdings.md) | 录入、查看、维护我的所有基金持仓 | 录入交易 / 看汇总 / 看已卖出 / 看切换链路 |
| [交易计划中心](./trade-plans.md) | 配置加仓 / 定投 / 纳指 100 套利等策略 | 新建计划 / 启停计划 / 看下一次定投日 |
| [基金切换收益分析](./fund-switch.md) | 分析「如果从 A 切到 B，到底是赚还是亏」 | 上传两边截图 → 校准价格 → 看 5 个数字卡 |
| [交易历史](./history.md) | 看计划自动生成的历史记录（只读） | 看累计投入 / 看覆盖了哪些策略 |
| [通知设置](./notify.md) | 把计划触发、持仓涨跌推送到手机 | 配 iOS Bark / Android 配对 / 同步计划 |
| [数据同步](./backup.md) | 跨浏览器、跨设备搬数据 | 配 WebDAV / 上传 / 恢复 / 下载本地备份 |

## 通用约定

- **数据全部存在浏览器本地** localStorage 里。换电脑、清缓存、换浏览器都会丢，**先去[数据同步](./backup.md)备份**。
- **应用不会替你下单**。所有「执行」「触发」都是提醒和记账，真正买入还是要去基金 APP / 券商操作，操作完回[持仓总览](./holdings.md)录入流水。
- **遇到看不到的删除/编辑按钮**，先看下面这条规则：聚合视图（基金汇总、已卖出、切换链路）都是从「成交流水」算出来的，**要改记录请去对应数据源改**。
