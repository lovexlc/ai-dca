上半 · 4 入口按钮风格选型
现状 PC 上 4 入口 grid 是 py-4 text-sm，单个 56px+ 高占满宽，偏移动端。以下 4 种 PC 原生紧凑写法供选：
选项 1 · Pill chip（顶推） · inline icon + 文字 · h-8 round-full · 靠左
收益明细 清仓分析 持仓分析 交易记录
优：PC 质感最好，与 Linear / GitHub UI 相似；与 hero 区轻分隔。
选项 2 · 纯 icon 圆按钮 · 36×36 · hover tooltip
优：最紧凑，适合放 hero 右上角；劣：依赖 tooltip，文字识别成本高。
选项 3 · 文字 tab · 无边框 · 活动项有下划线
收益明细 清仓分析 持仓分析 交易记录
优：子页导航质感，与详情页 tab 一致；劣：表达不是“入口”而是“页内切换”。
选项 4 · 缩小的 grid · 单个高 44px · icon 在上 + label 在下
收益明细 清仓分析 持仓分析 交易记录
优：与现有 grid 结构接近，变动最小；劣：44px 仍比 Pill chip 高。
