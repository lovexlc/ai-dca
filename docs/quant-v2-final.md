## 🚀 使用指南

### 1. 访问页面

部署后访问：
```
https://tools.freebacktrack.tech/?nav=quant-v2
```

或本地开发：
```
http://localhost:5173/?nav=quant-v2
```

### 2. 配置策略

1. 点击 **策略配置** Tab
2. 输入 H ETF（如 159513, 513100）
3. 输入 L ETF（如 159501）
4. 设置规则A阈值（如 3%）
5. 设置规则B阈值（如 1%）
6. 勾选"使用V2回测引擎"
7. 点击 **保存并运行回测**

### 3. 查看回测结果

1. 自动切换到 **回测分析** Tab
2. 顶部显示核心指标（收益、胜率、夏普、回撤）
3. 切换视图查看不同图表
4. 查看详细指标网格

### 4. 监控实盘

1. 点击 **实盘监控** Tab
2. 点击 **刷新** 按钮获取最新数据
3. 查看实时信号、持仓、行情

### 5. 查看交易

1. 点击 **交易历史** Tab
2. 查看所有模拟交易记录
