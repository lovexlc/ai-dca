# 美股策略助手主入口计划

## 目标

把用户侧名称从 `ai-dca` 调整为“美股策略助手”，新增默认主入口“策略指南”，让新用户先理解产品、生成演示数据、配置手机通知，再进入持仓/交易计划等功能。

## 本期范围

- 新增 `strategy` 顶级 tab，并作为默认入口。
- 用户可选择默认主页，本期保存到本地。
- 新增策略指南页面：产品说明、免责声明、通知配置图文说明、演示数据、全站 README、策略体系说明。
- 一键生成随机演示数据，并提供全局清除入口。
- 通知配置支持粘贴完整 Bark/Android 链接，由前端自动解析有效 Key/ID。
- 用户侧品牌统一为“美股策略助手”。

## 免责声明

部分策略内容由公开的金渐成公众号文章整理总结而来。本工具与金渐成本人及其公众号无官方关联、无授权关系，也不构成投资建议。投资有风险，请独立判断并自行承担决策结果。

## 验证

- 无 tab 参数默认进入策略指南或用户保存的默认主页。
- `?tab=` 显式参数优先。
- 策略指南 CTA 可跳转各 tab。
- 演示数据可生成、全局提示条出现、可清除。
- Bark 完整 URL 可解析出 Device Key。
- Android 完整测试 URL 可解析出 `android-` ID。
- 目标文件 lint 通过。
